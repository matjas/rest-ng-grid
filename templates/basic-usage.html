Basic usage

restNgGrid is RESTful angular(1.x) grid with no jQuery dependencies. It allow nest one grid in another and make dependencies make dependencies between parent and child.

How to use it?

Follow these steps

  1. include restNgGrid in script and css

  2. include restNgGrid as a dependency in your app

  3. add array data to $scope

  4. create options to grid. Fields array contains displayed parameters from data. Titles refers to columns name.

  5. use the rest-ng-grid directive and assign data "items" attribute
