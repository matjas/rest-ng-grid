Binding to remote data

restNgGrid supports data binding display either from local or remote data storage

For remote data binding you need to specify URL in dataSource.transport.query parameter for options attribute of the grid

In this example rest-ng-grid fetches data using URL defined in dataSource.transport.query parameter. To fill the grid set get-data-trigger expression to be true.
