rest-ng-grid

rest-ng-grid is a dat grid module for AngularJS(1.x). This is native AngularJs implementation, no jQuery. Module allows display and manage data from different resources, create parent-child grid with different columns name and different sources.

Module is during development, don't use in commercial projects
